We use CSS to styling webpages. We can styling any tags by identifying HTML tags, id name, class name, etc. which is called selector
For Example:
  tag_name{}
  .class_name{}
  #id_name{}
  *{}
  tag1_name,tag2_name{}

Property Description
background: transparent; To change background color or image or position or size , etc. (If we use this property for one time only).
background: rgba(100, 100, 100, 0.5); To change background color by mentioning rgba color model.
background-image: url("logo/bg.png"); To set background image by mentioning image path.
background-size: 50% 100%; To set heigth and width.
background-color: #53277b; To change background color.
background-attachment: fixed; To set background attachment(image or anything) position.
position: relative; To set position in fixed, relative, etc.
color: white; To set text color by mentioning color name.
text-shadow: 1px 1px 2px black, 0 0 25px blue, 0 0 5px darkblue; To set text shadow and it's color & direction.
float: left; To set orientation.
padding: 10px; To set internal margin by metioning pixel unit or percentage.
width: 25%; To set width by metioning percentage or pixel unit.
animation-duration: 10s; To set animation duration.
border: 3px solid cyan; To set border size, type & color.
box-sizing: border-box; To set border of content with style.
